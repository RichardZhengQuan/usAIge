# Usage HUD Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a native macOS floating HUD that displays every quota bucket exposed by the local Codex app-server, including percentage remaining and reset countdown.

**Architecture:** A SwiftUI executable is hosted in a borderless AppKit panel. A process-backed JSON-RPC client talks to `codex app-server`, a usage store normalizes and publishes quota snapshots, and isolated settings and visibility controllers keep UI behavior independent from transport details.

**Tech Stack:** Swift 6.2, Swift Package Manager, SwiftUI, AppKit, Foundation `Process`/`Pipe`, Observation, Swift Testing, macOS 15+

## Global Constraints

- Use the documented local Codex app-server as the only account and usage source.
- Do not scrape browser content, inspect screen pixels, estimate unsupported limits, or persist OpenAI credentials.
- Provide no menu-bar extra and no separate ChatGPT sign-in flow.
- Default to the bottom-right of the primary display and remember a safe position per display.
- Keep the panel above ordinary windows without stealing focus.
- Show a label, percentage remaining, and reset countdown for each supported bucket.
- Prefer app-server notifications over polling and keep idle CPU usage negligible.
- Hide for configured full-screen and privacy-sensitive activity using public macOS state only.

```swift
// Sources/UsageHUD/Domain/QuotaSnapshot.swift
import Foundation

struct RateLimitBucket: Codable, Equatable, Sendable {
    let limitID: String
    let limitName: String?
    let usedPercent: Double
    let windowDurationMinutes: Int?
    let resetsAt: TimeInterval?
    let planType: String?

    enum CodingKeys: String, CodingKey {
        case limitID = "limitId", limitName, usedPercent
        case windowDurationMinutes = "windowDurationMins"
        case resetsAt, planType
    }
}

struct QuotaSnapshot: Identifiable, Codable, Equatable, Sendable {
    let id: String
    let displayName: String
    let usedPercent: Double
    let remainingPercent: Double
    let resetAt: Date?
    let windowDurationMinutes: Int?
    let planType: String?
    let updatedAt: Date

    static func make(from bucket: RateLimitBucket, updatedAt: Date) -> Self {
        let used = min(100, max(0, bucket.usedPercent))
        let fallback = bucket.limitID
            .split(separator: "_")
            .map { $0.prefix(1).uppercased() + $0.dropFirst() }
            .joined(separator: " ")
        return .init(
            id: bucket.limitID,
            displayName: bucket.limitName?.isEmpty == false ? bucket.limitName! : fallback,
            usedPercent: used,
            remainingPercent: 100 - used,
            resetAt: bucket.resetsAt.map(Date.init(timeIntervalSince1970:)),
            windowDurationMinutes: bucket.windowDurationMinutes,
            planType: bucket.planType,
            updatedAt: updatedAt
        )
    }
}
```
